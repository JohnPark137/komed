## Notes

- Prefer the top-level curated mirror when you want the cleaner, intentionally selected `동의보감` corpus.
- Prefer the official-downloads bundle when you want the full direct-download artifacts, PDFs, or broader local reference copies.
- For the full imported `동의보감`, start with `dongui-bogam-ko-chapter-index.md`.
